This project was built to solve the need for a unified platform where engineers can showcase their work and knowledge.

## Features

- **Blog System**: Markdown-based blogging with categories and tags.
- **Project Showcase**: Rich project cards and details.
- **Technical Wiki**: A place for quick notes and snippets.
- **Theme Toggle**: Support for light and dark modes.
- **Responsive**: Fully mobile-first design.
- **Math Support**: High-performance LaTeX rendering.
- **Glassmorphism**: Elegant, modern UI with frosted glass effects.
