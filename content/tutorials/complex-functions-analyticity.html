Complex Functions & Analyticity

Problem Statement

(a) Let $u(x,y) = x^2 - y^2$ for $(x,y) \in \mathbb{R}^2$.
i. Prove that $u$ is a harmonic function. ii. Find an analytic function $f$ such that $\text{Re}(f) = u$ and $f(0) = 0$. iii. Use part (ii) to prove that $f(z) = z^2$.
Detailed Solution

Part (i): Harmonic Proof

A function $u(x,y)$ is harmonic if it satisfies Laplace's equation: $\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$.

First Derivatives

$\frac{\partial u}{\partial x} = 2x$, $\frac{\partial u}{\partial y} = -2y$

Second Derivatives

$\frac{\partial^2 u}{\partial x^2} = 2$, $\frac{\partial^2 u}{\partial y^2} = -2$

Sum: $2 + (-2) = 0$. Thus, $u$ is harmonic.

Part (ii): Finding the Analytic Function

Using Cauchy-Riemann equations: $\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}$ and $\frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$.

  1. Integrate $\frac{\partial v}{\partial y} = 2x$ with respect to $y$:
    $v(x,y) = 2xy + h(x)$
  2. Differentiate with respect to $x$: $\frac{\partial v}{\partial x} = 2y + h'(x)$.
  3. Equate to $-\frac{\partial u}{\partial y}$: $2y + h'(x) = 2y \implies h'(x) = 0$.
  4. Integrating gives $h(x) = C$. Thus $v = 2xy + C$.
  5. Apply $f(0)=0$: $(0^2-0^2) + i(2\cdot0\cdot0 + C) = 0 \implies C=0$.

Result: $f(z) = (x^2 - y^2) + i(2xy)$

Part (iii): Proof of Identity

Recall $z = x + iy$. Then:

$z^2 = (x+iy)^2 = x^2 + 2ixy - y^2 = (x^2-y^2) + i(2xy)$

Comparing with the result from part (ii), we confirm $f(z) = z^2$.

Fundamental Method

  • Harmonic Verification: Compute $\nabla^2 u$ and check if it equals zero.
  • Analytic Construction: Systematically use Cauchy-Riemann equations and integration of partials. Always remember to solve for the arbitrary function $h(x)$ and constant $C$.
  • Boundary Application: Use given initial values like $f(0)=0$ to finalize the constant of integration.

Complex Integration

Problem Statement

(b) Compute $\oint_C \frac{3z+1}{(z-1)^3(z+4)}dz$, where $C : |z|=3$.

Fig 1.0: Singularities relative to the contour |z|=3

Detailed Solution

Singularities

  • $z_1 = 1$ (Pole order 3)
  • $z_2 = -4$ (Simple Pole)

Analysis of $C$

Contour $C$ is $|z|=3$.

$z_1=1$ is Inside (|1| < 3)

$z_2=-4$ is Outside (|-4| > 3)

Application of Cauchy's Integral Formula

Formula for derivatives: $\oint_C \frac{f(z)}{(z-a)^{n+1}}dz = \frac{2\pi i}{n!} f^{(n)}(a)$.

Let $f(z) = \frac{3z+1}{z+4}$, with $a=1$ and $n=2$.

1. First Derivative: $f'(z) = \frac{3(z+4) - (3z+1)}{(z+4)^2} = \frac{11}{(z+4)^2}$

2. Second Derivative: $f''(z) = -22(z+4)^{-3} = \frac{-22}{(z+4)^3}$

3. Evaluation: $f''(1) = \frac{-22}{5^3} = \frac{-22}{125}$

Final Integration Calculation

$I = \frac{2\pi i}{2!} \cdot \left(\frac{-22}{125}\right) = \mathbf{-\frac{22\pi i}{125}}$

Fundamental Method

  • Singularity Check: Always sketch or calculate the distance of poles from the origin to check contour containment.
  • Order Definition: If a factor is $(z-a)^k$, the derivative order needed is $n = k-1$.
  • Function Selection: $f(z)$ must contain all parts of the integrand except the pole factor being evaluated.
  • Precision: Beware of sign errors during Quotient Rule differentiation.
